Feed-forward Neural Network for Emotion Classification

Architecture Summary

Input: 768-dimensional embeddings from WavLM-base (or 1024 for HuBERT-large)
Hidden Layer 1: 256 units with ReLU activation for feature learning
Hidden Layer 2: 128 units with ReLU activation for higher-level representations
Dropout: 50% dropout during training to prevent overfitting
Output: 4 classes (neutral, sadness, happiness, anger) or 6/7 classes depending on dataset
Total Parameters: ~200K trainable parameters
